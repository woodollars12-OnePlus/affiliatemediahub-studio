## 🎙️ Becky — Live Voice AI Coach
- Continuous voice conversation — stays on until you end it
- Speaks responses out loud using device speech synthesis
- 15-minute coaching drills for job interviews, sales pitches, affiliate marketing
- Powered by Google Gemini AI

## 🎬 Video Production Stack
- **Grok 4** — AI anchor video generation (Susan, Alisha, Lisa, Paul)
- **Sora** — OpenAI cinematic video generation
- **Google Veo 3.1** — Long-form consistent character videos
- **Google Flow** — Scene stitching and character consistency
- **InVideo AI** — Script-to-video automation
- **Canva AI** — Thumbnails and lower thirds
- **CapCut / VN Editor** — Mobile video editing

## 💻 Technical Stack
- **Frontend**: React (Babel CDN) — single HTML file
- **AI**: Google Gemini API (gemini-2.0-flash-preview)
- **Voice**: Web Speech API (SpeechRecognition + SpeechSynthesis)
- **Storage**: localStorage for API key and affiliate links
- **Hosting**: GitHub Pages (free)
- **Cost to build and run**: $0
